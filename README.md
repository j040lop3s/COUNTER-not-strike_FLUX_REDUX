# Contador React com Arquitetura Flux-Redux

Nesta aplicação, crio um componente que consome um estado global para atualizar o valor do contador exibido na página através de seus botões de incremento e decremento. Mesmo que seja um projeto simples, simulo um ambiente de aplicabilidade real com a separação de arquivos para cada parte da implementação da arquitetura. A estilização é feita com CSS puro personalizado com JS para criar um efeito visual.

# Tecnologias utilizadas

## Frontend

React (Vite): Para uma interface rápida e componentes reutilizáveis.
React-Redux: biblioteca que permite utilizar as funcionalidades redux.

## Arquitetura Flux

Foi implementada através de actions, reducers, store.
